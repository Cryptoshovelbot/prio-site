## Plan d'action — 3 priorites

### Priorite 1 : Ajouter hreflang sur toutes les pages multilingues

**Impact** : Eviter la cannibalisation entre versions linguistiques, meilleur ciblage geo.

Pour chaque groupe de pages equivalentes, ajouter dans le `<head>` :
```html
<link rel="alternate" hreflang="en" href="https://getprio.io/blog/how-to-find-apartment-barcelona-2026/">
<link rel="alternate" hreflang="es" href="https://getprio.io/blog/como-encontrar-piso-barcelona-2026/">
<link rel="alternate" hreflang="fr" href="https://getprio.io/blog/trouver-appartement-barcelone-2026/">
<link rel="alternate" hreflang="x-default" href="https://getprio.io/blog/how-to-find-apartment-barcelona-2026/">
```

**Fichiers concernes** : 4 groupes de 3 articles + homepage + pages villes = ~16 fichiers.

### Priorite 2 : Ajouter og:image sur toutes les pages

**Impact** : Meilleur CTR social, eligibilite Google Discover.

Minimum : reutiliser `/assets/og-image.png` sur toutes les pages. Ideal : une image unique par article de blog.

Pour chaque fichier HTML dans le `<head>` :
```html
<meta property="og:image" content="https://getprio.io/assets/og-image.png">
<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:image" content="https://getprio.io/assets/og-image.png">
```

**Fichiers concernes** : 38 fichiers index.html (tous sauf homepage et NIE guide).

### Priorite 3 : Deployer FAQPage schema sur les articles de blog

**Impact** : Rich snippets FAQ dans Google = espace SERP accru, meilleur CTR.

Pour chaque article qui a une section FAQ ou du contenu Q&A, ajouter :
```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "...",
      "acceptedAnswer": { "@type": "Answer", "text": "..." }
    }
  ]
}
</script>
```

**Pages prioritaires** (contenu Q&A existant a formaliser) :
- `/blog/how-to-find-apartment-barcelona-2026/`
- `/blog/idealista-scams-how-to-spot-fake-listings/`
- `/blog/documents-rent-apartment-spain/`
- `/blog/idealista-vs-fotocasa-vs-badi-comparison/`
- `/blog/how-long-find-apartment-barcelona/`
- Toutes les pages villes ES sans FAQ

### Bonus : Reformuler des H2 en questions

Les headings en format question matchent mieux les requetes conversationnelles et featured snippets :
- "The State of Barcelona's Rental Market" -> "What is the state of Barcelona's rental market in 2026?"
- "Where to Search: Platforms Compared" -> "Which platforms should you use to find an apartment in Barcelona?"
- "Documents You'll Need" -> "What documents do you need to rent in Spain?"
